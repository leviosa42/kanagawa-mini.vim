## :ocean: kanagawa-mini.vim :ocean:

A Minimal color scheme ported from [rebelot/kanagawa.nvim] for Vim.

## Configure

```vimscript
" Default options.
let g:kanagawa_mini = {
    \ 'undercurl': v:true,
    \ 'commentStyle': 'italic',
    \ 'functionStyle': 'NONE',
    \ 'keywordStyle': 'italic',
    \ 'statementStyle': 'bold',
    \ 'typeStyle': 'NONE',
    \ 'specialReturn': v:true,
    \ 'specialExeption': v:true,
    \ 'transparent': v:false,
    \ 'dimInactive': v:false,
    \ 'terminalColors': v:true,
    \ 'theme': 'default'
    \ }
```
